My Portfolio

Fast Spec-Tree Renaming

Fast Spec-Tree Renaming is a productivity tool for CATIA that enables rapid and consistent renaming of the specification tree.

It allows users to apply custom naming conventions without manually renaming each part or feature. By automating repetitive tasks, it improves model readability and helps enforce clear and scalable design standards.

Generative BOM

Generative BOM for large CATIA assemblies using CAA interfaces. Independent spare parts lists with vendor codes, specs, images, and custom attributes.

Image-based tables for internal catalogs, documentation or commercial components. Export to XLS, CSV, or PDF for external manufacturing workflows.

AGV Layout Simulation

Simulation developed in CATIA to validate the motion feasibility of an AGV within a constrained environment.

This initial validation enables moving forward with deeper analyses, such as optimization, cycle time evaluation, and design refinement.
