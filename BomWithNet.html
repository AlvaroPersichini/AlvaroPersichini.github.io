BOM with .NET

Automated Bill of Materials

This solution allows to quickly deploy a customized BOM with some worhty features. Its structure combines interfaces from various branches of the CATIA CAA infrastructue's tree. Suited for large assemblies.

Spear parts list option

A separated spare parts list can be obtain usgin the part/assembly properties including vendor codes, data sheets, images or any other atribute that may be needed.

Deploy a list of images for catalogs

A table of images and properties can be created to be used in catalogs or internal files, fully customized and rapid integration.

Export To: .xls, .cvs or .pdf :

Beside of using the BOM for a native drawing, the application is able to create and export a file with several formats for external use.
